Release 0.10.0 – 2016-09-30

Type Changes By
Performance: change Package to Profile in the SysML 1.4 OMG profile :warning: The profile needs to be re-apply: - Open your Model - Select the root in the Model Explorer - Select Profile Properties view - SysML 1.4 profile is applied - Re-apply SysML 1.4 profile => All SysML 1.4 sub profile should also be applied Fixes 501077. bmaggi
Many constraints have added from the official norm. You can validate your model by: - Opening the model in model explorer - Right click on any element - Chose Validation > Validate Model - The result of the validation is available in bmaggi
Add Association Block element, an AssociationBlock is an association class stereotyped by Block Fixes 478268. bmaggi
All the fixes can be found here: https://bugs.eclipse.org/bugs/buglist.cgi?classification=Modeling&product=Papyrus&component=SysML&target_milestone=0.10.0
