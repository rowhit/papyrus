About Designer Languages JAVA

This module contains the different plugins relative to Papyrus Designer Java code generation and reverse

Project Modules

This project has declared the following modules:

Name Description
org.eclipse.papyrus.designer.languages.java.codegen This module contains a Java code generator
org.eclipse.papyrus.designer.languages.java.codegen.ui This module contains the UI part of the Java code generator
org.eclipse.papyrus.designer.languages.java.jdt.project This module contains a JDT integration into Papyrus
org.eclipse.papyrus.designer.languages.java.library This module contains a library of Java basic types
org.eclipse.papyrus.designer.languages.java.library.ui This module contains the UI part of the Java library
org.eclipse.papyrus.designer.languages.java.profile This module contains a Java profile
org.eclipse.papyrus.designer.languages.java.profile.ui This module contains the UI integration of the Java profile
org.eclipse.papyrus.designer.languages.java.reverse This module contains the feature to reverse Java code into a UML model
org.eclipse.papyrus.designer.languages.java.reverse.ui This module contains the UI of the Java code reverse feature
